#  yolov8模型之訓練與資料補強分析

一、研究背景與動機
本專題主題以美甲（Nail）影像辨識為核心，目標為建立一套能夠自動偵測指甲區域之物件偵測模型，作為後續美甲分析、設計輔助或智慧美甲應用的基礎。
在初期模型訓練與測試過程中發現，模型對於有彩繪、顏色明顯的美甲影像辨識效果良好，但在實際測試時，對於裸甲（未上色指甲）影像的辨識能力明顯不足，因此進一步針對資料集進行補強與再訓練。
________________________________________
二、資料集與補強策略說明
1. 原始資料集特性:影像內容以美甲指甲為主、指甲顏色、圖樣明顯，背景多樣、類別設定：	nail（指甲）、	finger（手指）
2. 問題觀察（測試階段）
在模型測試過程中發現：
(1)對於裸甲影像（接近膚色、對比低）辨識率偏低
(2)容易將裸甲與手指背景混淆
(3)顯示模型對於「低對比指甲特徵」學習不足
3. 補強資料策略
為改善上述問題，本研究採取以下措施：
(1)新增裸甲影像資料
(2)由人工方式重新蒐集裸甲影像
(3)使用 MakeSense.ai 進行精確框選標註
(4)補強裸甲在不同光線、角度、膚色條件下的樣本數

目的並非改變主題，而是提升模型在美甲整體情境下的泛化能力
________________________________________
三、模型訓練設定
•	模型架構：YOLOv8
•	訓練方式：From scratch 訓練
•	影像尺寸：640 × 640
•	Batch size：16
•	Epochs：100
•	資料格式：YOLO Detection format
________________________________________
四、訓練結果與數據分析
1. 訓練收斂情形
由訓練曲線可觀察到：
(1)train/box_loss、train/cls_loss、train/dfl_loss 持續下降
(2)模型成功學習指甲在不同外觀條件下的空間位置與結構特徵
顯示補強後資料能有效參與模型學習。

2. 驗證集效能表現
   (1)Precision 約 0.70
   (2)Recall 約 0.75
   (3)mAP@0.5 約 0.74
   (4)mAP@0.5:0.95 約 0.26–0.28
結果顯示模型在一般美甲情境下具備穩定辨識能力，而裸甲樣本的加入有助於降低誤判情形。

3. 裸甲資料補強成效分析
   (1)裸甲樣本加入後，模型對「低顏色對比指甲」的召回率明顯改善
   (2)減少裸甲被誤判為 finger 類別的情況
   (3)提升模型在實際應用場景中的可靠性
________________________________________
五、結果討論
本研究結果顯示，資料分佈對物件偵測模型的影響極為顯著。
雖然研究主題為美甲，但裸甲在實際使用場景中同樣存在，若資料集中裸甲比例不足，模型容易產生偏誤。
因此，本研究透過裸甲資料補強，使模型能同時適應：
1.彩繪美甲
2.純色指甲
3.裸甲（自然指甲）
使模型更符合真實美甲應用需求。
________________________________________
六、結論
本專題以美甲影像辨識為核心，並透過實驗過程發現裸甲為模型辨識上的關鍵挑戰。
經由針對裸甲影像進行資料補強與再訓練後，模型整體辨識表現獲得明顯改善，提升了系統在實際應用情境中的穩定性與泛用性。
